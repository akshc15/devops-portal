01

Create a new GitHub account (or use existing)

Go to github.com → Sign up → choose a username. This username becomes your URL. Pick something professional like yourname-devops or keep your existing username.

02

Create the magic repository

Click + (top right) → New repository → Repository name: type exactly yourusername.github.io (replace yourusername with your actual username) → tick Public → tick Add a README file → click Create repository. This exact naming is what activates GitHub Pages.

03

Upload all 3 files

Go to your new repo → click Add file → Upload files → drag all 3 files: index.html, devops-bible.html, 30day-challenge.html → scroll down → click Commit changes.

04

Enable GitHub Pages

In your repo → click Settings tab → left sidebar scroll to Pages → Source: Deploy from a branch → Branch: main → folder: / (root) → click Save. Wait 60 seconds.

✓ Your site is live at: yourusername.github.io
05
